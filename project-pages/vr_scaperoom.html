VR ScapeRoom
Basic escape room game in virtual reality for a final university project.
Project Details / Background
Final project for the Virtual and Augmented Reality course, where the goal was to create an escape room. My team and I decided to create this mini-game where an experiment goes wrong, and your objective is to return to your time. I contributed to this project as a mechanics programmer and mechanics designer.
